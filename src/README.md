## 🚀 Key Features

### 1. **Friend Management**
- Add and view detailed friend profiles
- Track "Days Since Last Contact"
- Set custom connection goals
- Visual status indicators (On Track, Almost Due, Overdue)

### 2. **Interaction Timeline**
- Log different types of interactions:
  - 📞 Calls
  - 💬 Text Messages
  - 🎥 Video Calls
- Filter timeline by interaction type
- Clean and beautiful activity history

### 3. **Friendship Analytics**
- Pie Chart showing conversation distribution
- Track your consistency with friends
